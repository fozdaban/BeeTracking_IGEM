Bee Behavior Tracking System

This project uses computer vision to track and quantify the movement of 5–10 bees inside an artificial hive. By combining YOLOv8 detection with multi-object tracking (ByteTrack / OC-SORT), the system generates behavioral metrics such as speed and activity levels.

The purpose is to compare control and treatment groups to determine whether engineered gut microbiome modifications produce measurable changes in bee behavior.
